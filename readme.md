## estkpm-cache

caching js or jsx script with [estkpm](https://github.com/nbqx/estkpm)(extendscript package manger)

## Install

    $ git clone https://github.com/nbqx/estkpm-cache.git
    $ cd estk-cache
    $ npm install -g .

## Usage

    $ estk-cache sync -c /path/to/package.jsx
    $ estk-cache list
    $ estk-cache flush

see also

    $ estk-cache -h
